# AI Ticket Support Platform

## Features

- Ticket Management
- AI Ticket Triage
- Knowledge Base
- Vector Search
- AI Reply Generation
- Email Notifications
- Operations Analytics

## Tech Stack

Frontend:
- React
- Vite
- Supabase

Backend:
- FastAPI
- PostgreSQL
- SQLAlchemy

AI:
- Groq
- pgvector
- RAG

## Installation

Frontend:
npm install
npm run dev

Backend:
pip install -r requirements.txt
uvicorn main:app --reload
